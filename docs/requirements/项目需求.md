# 卡片管理 App 功能点清单（纯本地版，仅功能范围）

以下版本仅描述**功能点**，不区分实现优先级，不展开 UI/UX 方案，不讨论合规与技术实现限制。
产品前提为：

* 无 backend
* 无用户登录
* 所有数据仅保存在本地
* 依靠设备通行密钥/本地安全能力进行加解密
* 支持银行卡、会员卡、证件管理
* 支持银行卡支付必要信息的保存与复制

---

# 一、产品基础功能

## 1. 应用初始化

* 首次启动初始化本地数据空间
* 首次启动创建本地安全保护机制
* 首次启动生成本地加密数据容器
* 首次启动引导用户设置查看敏感信息的保护方式
* 首次启动写入内置模板资源
* 首次启动写入默认分类与默认标签
* 首次启动写入默认提醒规则

* 检测本地数据是否损坏
* 检测本地模板资源是否完整
* 检测当前设备是否支持生物识别/设备通行密钥
* 支持重新初始化本地数据空间

## 2. 应用解锁与访问控制

* 应用启动时解锁
* 应用切回前台时解锁
* 敏感信息查看前再次验证
* 复制敏感信息前再次验证
* 导出备份前再次验证
* 恢复备份前再次验证
* 修改安全设置前再次验证
* 支持生物识别验证
* 支持 PIN 验证
* 支持设备安全认证失败后的重试
* 支持连续失败后的短时锁定
* 支持自动锁定
* 支持后台切换后自动锁定
* 支持一段时间无操作后自动锁定

* 支持敏感页面退出后重新隐藏明文信息
